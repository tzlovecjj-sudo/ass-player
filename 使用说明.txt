ASS字幕播放器 - Windows 使用说明

【快速开始】
1. 双击运行 "start.bat"
2. 等待浏览器自动打开播放器页面
3. 开始使用！

【系统要求】
- Windows 7/8/10/11
- Python 3.7 或更高版本
- 现代浏览器（推荐 Chrome、Edge）

【首次运行】
- 首次运行会自动安装 Flask 依赖
- 如果遇到防火墙提示，请允许访问
- 如果浏览器没有自动打开，请手动访问: http://127.0.0.1:5000

【功能特点】
✓ 本地视频播放（MP4、WebM、AVI等格式）
✓ ASS字幕文件完整支持
✓ 在线视频播放（B站、YouTube等）
✓ 全屏播放模式
✓ 键盘快捷键控制

【键盘快捷键】
空格键 - 播放/暂停
ESC    - 退出全屏
← →    - 前进/后退5秒

【常见问题】
1. 如果提示"未找到Python"，请安装Python并确保勾选"Add to PATH"
2. 如果端口被占用，请关闭其他占用5000端口的程序
3. 如果视频无法播放，请尝试使用Chrome浏览器

【文件结构】
确保以下目录和文件存在：
- templates/ (包含HTML文件)
- static/css/ (包含CSS样式文件)  
- static/js/ (包含JavaScript文件)

【技术支持】
如有问题，请查看命令行中的错误信息或联系开发者。
